# Kepanjen School Buffer Zone (KSBZ)



##### KSBZ

##### Project language: English



##### About

KSBZ project is about to indentify how closer a school from another one in Kepanjen district at Malang regency, East Java.



##### Result

!\[Result](result/buffer\_result.png) 

!\[Result](./result/buffer\_result.png)



##### Context

This project identify every education building in the regency from elementary school to university. Identifying buffer zones is essential for analyzing how close one building is to another. The purpose of buffer zone analysis is to determine whether a building within a certain radius can be easily accessed by all residents in the area.

##### Methods

###### Map explanation 

* The buffer zone used is a one-kilometer radius in a circular pattern centered on the building with the legends are already on the map. 
* Project located in Kepanjen region, Malang regency, East Java, Indonesia. 
* Number of school 40 including elementary school to university
* Projection EGS 84 / Pseudo Mercator and EPSG:3857 as a CRS

##### Results

There are fourty points were identified including elementary school to university in Kepanjen. The majority of those points are close to each other making it too dense on some areas. **In my view, this uneven distribution makes it difficult for people living far from the city centre to access schools.**
